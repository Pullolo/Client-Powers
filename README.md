# ClientPowers - Fabric Client Mod
This mod adds 13 visual superpowers with unique HUD effects, particles, and combat awareness — all client-side, no server required.

### Notes
- Not all powers are equal, some are more impactful than others.
- Mod was made and tested for 1.21+ versions of Minecraft. Issues on older versions will not be addressed.
- All effects are purely client-side and visual — no gameplay mechanics are modified on the server.

### Impact References
- *Small Impact 💚*
- *Medium Impact (Noticeable visual changes) 💛*
- *Large Impact (Changes how you perceive the game) 💙*

### Current list of Powers and their impact
- 💛 - Flame 🔥 *(fire particles, ember trail, brightness boost)*
- 💚 - Frost ❄️ *(snowflake overlay, cryo particles)*
- 💛 - Thunder ⚡ *(screen flash, speed lines while sprinting)*
- 💙 - Void 🌑 *(entity glow through walls, portal finder compass)*
- 💙 - Stargazer ⭐ *(starfield overlay, shooting stars, moon phase HUD)*
- 💚 - Nature 🌿 *(animal glow, falling leaf overlay)*
- 💛 - Shadow 🌒 *(dark vision, entity glow, shadow wisps)*
- 💚 - Ocean 🌊 *(underwater clarity, bubble overlay)*
- 💛 - Blood 🩸 *(hostile mob glow, heartbeat vignette, blood drips)*
- 💛 - Wind 💨 *(wind streak overlay, weather forecast HUD)*
- 💚 - Crystal 💎 *(brightness boost, rainbow prism edge sparkles)*
- 💙 - Lodestar 🧭 *(ore compass, detects all ore types within 16 blocks)*
- 💙 - Ninja 🥷 *(player glow, crit window indicator, hit direction arc, blind spot alert, backstab detector)*

### Keybindings
- `R` — Open Power Wheel
- `P` — Open Settings
- `G` — Open Power Guide
